# Background music (optional)

To enable the floating music toggle, drop a royalty-free instrumental track here named:

```
audio/instrumental.mp3
```

Suggested: a soft piano or violin instrumental (30–90s, looped). Keep it small (< 2 MB) for fast loads.

If no file is present, the toggle still appears but shows a gentle hint instead of playing — nothing breaks.

Good free sources: Pixabay Music, Chosic, or the YouTube Audio Library (check the licence allows web use).
